# Fundación Patitas Felices

Maqueta web responsive que da a conocer la labor de rescate y adopción de animales.

## Estructura del proyecto

PRoyectoFinalHTMLYCSSBrayanEspinosa/
├── css/
├── docs/
├── html/
├── media/
└── index.html


## Ejecución

Abre `index.html` en tu navegador.<br>
No requiere instalación de dependencias.

[visualizala pagina](https://bespinosaosorio96-ship-it.github.io/ProyectoFinalHTMLYCSSESpionosaBrayan/)
